# Omrekenen

Een eenvoudige webapp voor leerlingen om inhoudsmaten stap voor stap te oefenen.

## Inhoud

- Lespresentatie met terug- en verderknoppen.
- Uitleg over L, dL, cL, mL.
- Koppelingen `1 cm3 = 1 mL` en `1 dm3 = 1 L`.
- Exit-ticket minigame met hulpschema.
- Finale ronde zonder hulpschema met 10 goede antwoorden op een rij.

## Scripts

```bash
npm install
npm run dev
npm test
npm run build
```
